# Kotlin Country Cards App

This is a small Android app made with **Kotlin** and **Jetpack Compose**.

The app shows information about **four countries** using Material Design
cards. Each card displays the **flag**, the **country name**, and the
**capital city**.

## Features

- A **Column** of 4 cards (no LazyColumn).
- Each card contains:
  - Image of the **country flag**
  - **Country name** (Geo font, bold)
  - **Capital city** text (Geo font)
- Customized card design:
  - Rounded corners
  - Soft background colors (primary/secondary container colors)
- Uses **Material 3** with:
  - **Dynamic color** support
  - **Dark theme** support
- App icon changed to a **globe** image.

Example countries used:

- Finland – Helsinki  
- Bangladesh – Dhaka  
- Japan – Tokyo  
- Italy – Rome
