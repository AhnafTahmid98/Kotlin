# Kotlin Primes and Evens

This is a simple Kotlin console program.

## What the program does

1. Finds the first 50 prime numbers and stores them in a list.
2. Prints the list of prime numbers.
3. Creates a list of the first 50 even numbers (2, 4, ..., 100).
4. Adds the even numbers to the prime list to create a combined list.
5. Prints the combined list.

## How to run

Copy `Main.kt` into https://play.kotlinlang.org/ and press **Run**
to get the same output that is saved in `output.txt`.
